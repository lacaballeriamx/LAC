Este es un pequeño experimento para un juego sencillo: presionar el signo de interrogación para mostrar una pregunta al azar e iniciar un contador.

Pitch del juego:
"¿Sin charla en la primera cita? ¿Compañeros de trabajo con tiempo que matar? ¿Hermanos de toda la vida o amigos desde hace años que juran saberlo todo sobre el otro? Con Tevaa, hoy seguro descubres algo nuevo sobre tu date, tus compañeros, tus amigos, familia.. ¡o hasta sobre ti mismo!

Sólo presiona el signo de interrogación (?) y deja que el diálogo fluya. Toma el tiempo que desees para pensar cada respuesta - y recuerda que no importa salirse de track, después de todo, lo más bonito es perderse en la conversación. Venga, ¡te va!"


No soy programador, y este es mi primer intento en algo así. No dudo que pueda mejorarse; de hecho quiero añadir un registro de cuánto tiempo tardó el usuario en cada pregunta, y recopilar el historial de ello para al último poderle mostrar cuánto tiempo tardó en cada pregunta.

Gracias por leerme.
